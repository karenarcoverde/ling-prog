Este programa possui um gerenciamento de pacientes hospitalares.

Instruções sobre o Menu:
O usuário deverá escolher uma opção das citadas abaixo. Ele poderá escolher quantas vezes quiser as opções enquanto não for escolhida a opção 4 (Sair).

1. Insere paciente -> Acrescenta um paciente no cadastro de acordo com o nome escolhido e o tipo de paciente (paciente normal, paciente número de consultas, paciente doença).
Para cada tipo de paciente escolhido, digite as seguintes informações:
=============================================================================
Para Paciente normal: 
O usuário deve apenas digitar o nome do paciente.
-----------------------------------------------------------------------------
Para Paciente Número de Consultas:
O usuário deve digitar o nome do paciente e o respectivo número de consultas que o paciente já fez no hospital.
-----------------------------------------------------------------------------
Para Paciente Doença:
O usuário deve digitar o nome do paciente e o nome da doença que o paciente está.
=============================================================================
2. Busca paciente -> Mostra as informações do paciente de acordo com o nome escolhido.
3. Exibe pacientes -> Mostra todos os pacientes e as suas respectivas informações.
4. Sair -> O programa finaliza.
     

Para compilar:
Digite make no terminal

Para executar:
./pacientes
